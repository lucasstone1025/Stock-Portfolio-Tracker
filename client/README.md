# Client

React frontend for the TrendTracker application, built with Vite.

See the [main README](../README.md) for full documentation.

## Quick Start

```bash
npm install
npm run dev
```

The development server runs at `http://localhost:5173` and proxies API requests to the backend.

## Scripts

- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
